Innovación

Sistema de asistencia

Sistema de asistencia
en la conducción ABS, ASR.

ABS

El Sistema de Freno Antibloqueo te permite controlar la trayectoria del auto en un frenado de emergencia.

ASR

El Control de Tracción mantiene la adherencia de las llantas en superficies deslizantes y con aceleraciones bruscas.

EBD

Este dispositivo distribuye la fuerza de frenado entre las llantas traseras y las delanteras, para detener el vehículo de manera eficiente.

ESC

El Sistema Electrónico de Estabilidad evita que el auto patine frenando automáticamente una o más llantas.

Presión de neumáticos

Vigila la presión de las llantas, para evitar variaciones peligrosas en tus viajes.

Sistema de asistencia

Multicolision brake

En caso de colisión, este sistema aplica los frenos y enciende las luces de emergencia para mantenerte seguro.

Detección de Fatiga

Si amas la carretera, este sistema es importante: Detecta tus hábitos de manejo y te avisa cuando debes hacer una pausa.

Sistema de seguridad (6 bolsas de aire)

El conductor y el copiloto cuentan con 6 bolsas de aire, frontales y laterales tipo cortina.

Habitáculo de pasajeros

Mayor rigidez para mayor seguridad. Touareg te mantiene seguro en caso de accidente.
